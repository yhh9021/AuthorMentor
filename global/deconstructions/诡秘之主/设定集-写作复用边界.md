# 设定集-写作复用边界：《诡秘之主.txt》

## 可复用结构

- **小人物入口承接超大世界**：开篇先设计一组可日常运转的人际关系和经济压力，再让超凡事件逐个破坏这些稳定点。
- **秘密组织让信息差成为持续爽点**：给组织一个安全会议机制、匿名身份、交换规则和成员现实任务，使每次会议都带来行动后果。
- **案件单元嵌套卷级主线**：为单元案设置表层凶手、资源线索和主线回扣，结案时至少改变一个长期关系或认知。
- **能力越强，代价越贴脸**：设计能力时同步写出触发条件、反制方式、滥用后果和对角色性格的侵蚀。
- **马甲分层解决长篇行动半径**：给每个马甲配置专属目标、语气、联系人和不可暴露的秘密，并定期让身份边界产生压力。
- **伏笔以道具、句子和人物债三层回收**：记录每个伏笔的表层用途、隐藏用途和回收时要付出的代价，避免只做名词重复。
- **仪式协议限制超自然便利**：为每个常用魔法写出输入、输出、权限、消耗和错误连接对象。
- **群像成员各自代表一块世界**：设计群像时先定义成员的世界接口，再设计私人欲望和与主角的交换关系。
- **社会苦难转化为超凡阴谋证据**：先写具体受害者和生活细节，再揭示这些苦难如何被高层利用。
- **反派同样是规则玩家**：为反派设计清晰目标、资源来源和规则优势，再让主角找到优势的边界。
- **恐怖与幽默交替缓冲高压阅读**：让幽默来自人物处境和信息差，不打断危机本身的后果。
- **晋升爽点绑定人格考题**：为每个等级设置一句可被剧情验证的扮演原则，并安排一次违反原则的诱惑。
- **地图升级同步改写题材类型**：每卷先确定题材承诺，再挑选最适合该题材的能力、敌人和社会矛盾。
- **终局胜利保留未完成使命**：终局后保留一个高阶问题，同时明确主角已完成本书的核心情感承诺。

## 不可复用边界

- **绯红自杀案把穿越者推上灰雾**：空间权柄不能立刻变成全能战力；每次使用都要受知识、灵性、身份暴露和仪式条件限制。
- **值夜者审讯转成官方庇护入口**：官方机构不能只做说明书；它必须在后续用牺牲、纪律和资源代价证明自身重量。
- **罗塞尔日记把魔药系统从升级改成表演**：扮演法只能加速消化，不能免除材料、仪式、精神污染和高位存在注视。
- **2-049木偶把低序列战斗写成规则求生**：规则战不能脱离角色认知；主角只能利用现场已展示的信息完成反杀。
- **梅高欧丝炸弹终结廷根日常**：牺牲不能只为煽情；必须改变主角身份、关系网和后续行动路线。
- **献祭赐予把塔罗会升级为跨空间物流**：仪式功能要有明确符号、咒文、祭品和对象，不宜用一句“神力完成”跳过过程。
- **兰尔乌斯复仇第一步连接东区苦难**：复仇对象不能孤立存在；他要和城市资源、组织利益、前卷伤口都有实质连接。
- **卡平别墅把黑皇帝姿态留给被救者**：匿名制裁要有调查成本和法律失灵前提，否则会变成任意私刑。
- **无面人晋升让身份系统成为战斗系统**：变脸不能等于换人生；知识、习惯、关系债务和心理锚仍会暴露身份。
- **贝克兰德大雾霾把都市阴谋推成历史伤口**：大灾难不能只做背景板；必须留下人口损失、角色创伤和制度性追责。
- **班西港把红天使余火投到海上冒险**：怪谈地标要给出可追溯历史和角色后续用途，不能只靠恐怖氛围制造一次性惊吓。
- **海神卡维图瓦死亡让愚者拥有地上锚点**：信徒不是纯电池；回应祈祷、保护社区和承担因果都要写进代价。
- **格罗塞尔游记用副本重组纪元史**：副本必须改变主线认知或资源结构，单纯插入番外式冒险会稀释节奏。
- **乔治三世线把黑皇帝仪式藏进国家工程**：政治阴谋要与既有政策、财政、战争和社会伤亡挂钩，否则成神仪式会显得悬浮。
- **因斯复仇终章用0-08清算廷根旧债**：复仇延期要持续给读者补充线索和阶段性推进，不能只在终局突然提起旧仇。
- **罗塞尔真相把穿越升级为旧日阴谋**：终极真相要回扣最初细节，不能用新设定强行覆盖旧问题。
- **乔治三世成神失败引爆公开战争**：多方协作要提前布置人脉、资源和目标分工，临时叫援会削弱可信度。
- **白银城走出神弃之地兑现太阳线**：救赎不能只写抵达新地图；必须处理迁徙、牺牲、信仰和融入成本。
- **门先生归来把救援变成阿蒙夺权陷阱**：反转要尊重早期诅咒与污染规则，不能把被救者临时写成工具。
- **源堡终局把成神胜负落回人性锚点**：神战不能只堆概念名词；读者必须看见主角最私人关系如何参与胜负。
- **序列途径是世界的职业阶梯**：阶梯越清晰，越要给每级设置独特代价和场景需求，避免沦为数字等级。
- **扮演法把升级变成自我管理**：扮演原则要从剧情事件中提炼，不能由角色口头宣告后直接升级。
- **灰雾与源堡是外挂也是陷阱**：外挂功能必须分阶段展示，且每个新功能都要引入新敌意或新责任。
- **塔罗会是秘密组织的产品化设计**：成员必须有独立欲望和行动能力，不能只作为主角信息终端。
- **锚把神明拉回人类社会**：锚点要通过具体信徒和共同经历呈现，空泛崇拜无法承载情感。
- **旧日与外神提供终局倒计时**：宇宙尺度设定要保留和主角私人命运的接口，不能只展示名词层级。
- **正神教会兼具庇护与垄断**：垄断不能写成纯反派；基层守护者的牺牲必须与高层博弈区分。
- **封印物把道具写成危险同事**：道具效果要提前说明或通过观察推断，结局才不会像临时开挂。
- **秘密组织构成地下资源市场**：组织要有独立目标、资源和行动风格，不能只换名字做同类反派。
- **超凡材料和配方制造稀缺经济**：价格和稀缺性要随序列提高而变化，不能让主角永远用同一种渠道解决问题。
- **非凡特性聚合让敌我都被命运拉近**：聚合不能替作者省略因果；角色仍要有主动行动和可追踪线索。
- **地图层级对应认知层级**：新地图不能切断旧情感线；每次离开都要带走债务、敌人或资源。
- **身份马甲是社会接口**：每个马甲都要有独立代价和不可跨用的信息，不能只是换称呼。
- **不可知信息本身会杀人**：污染规则要稳定，不能让角色随剧情需要忽然免疫或忽然脆弱。
- **仪式魔法是世界的API**：仪式细节不能无限膨胀；每个步骤都要服务风险、限制或反转。
- **工业贫困是邪神仪式的燃料**：不能把苦难只当能量设定；要通过老科勒、丽芙、黛西这类具体人物承载。
- **王权成神需要历史与制度配合**：国家级仪式必须有财政、战争、舆论和秘密工程痕迹支撑。
- **神弃之地保存被断裂的历史**：孤岛线要不断影响主线决策，不能与主角所在地图完全平行。
- **门途径把空间旅行写成星空风险**：空间能力必须有坐标、冷却、污染或追踪限制，否则会抹平地图难度。
- **复活不是重置，而是债务延续**：复活必须少用且每次改变局面，否则死亡威胁会失效。
- **纪元时间线让神话成为历史推理题**：历史反转要保留前文证据，不能靠权威角色一次性讲完。
- **灵界通信让距离保留戏剧性**：通信手段不能无条件即时万能；延迟、截获和回应代价要参与剧情。
- **镜子与问答物把信息优势变成羞耻成本**：强情报工具必须有明确代价和不能回答的边界。
- **小人物入口承接超大世界**：过早堆神名和体系会让读者失去代入；日常若不被主线改变，则会拖慢节奏。
- **秘密组织让信息差成为持续爽点**：主角若真的全知全能，紧张感消失；成员若只汇报消息，组织会变成读报会。
- **案件单元嵌套卷级主线**：单元案若不回扣主线，会像支线填充；主线若压过案件，会削弱侦探乐趣。
- **能力越强，代价越贴脸**：只有能力没有代价会破坏悬疑；代价只在说明中出现也难以让读者相信。
- **马甲分层解决长篇行动半径**：马甲若无成本，会削弱主角真实人格；马甲太多又缺乏标志性场景，会造成记忆负担。
- **伏笔以道具、句子和人物债三层回收**：回收若只靠解释会显得机械；伏笔太密且无阶段兑现会让阅读疲劳。
- **仪式协议限制超自然便利**：协议过细会像说明书；协议随剧情变动会让读者觉得规则失信。
- **群像成员各自代表一块世界**：角色只承担设定入口会工具化；只有私人戏又会拖慢主线。
- **社会苦难转化为超凡阴谋证据**：直接把苦难当设定燃料会显得冷漠；只写控诉又会脱离类型节奏。
- **反派同样是规则玩家**：反派若只靠作者保护，会让读者挫败；反派若频繁犯低级错，会失去压迫感。
- **恐怖与幽默交替缓冲高压阅读**：笑点若在牺牲现场抢戏，会削弱悲剧；长期无幽默则会让高压设定疲劳。
- **晋升爽点绑定人格考题**：原则若只被口头复述，会像修炼口诀；人格考题若太抽象，会拖慢战斗节奏。
- **地图升级同步改写题材类型**：题材切换若缺少旧线牵引，会像换书；若只重复前卷结构，会丧失长篇推进感。
- **终局胜利保留未完成使命**：悬念留太多会像断尾；全部清零又会削弱宏大世界的真实感。
