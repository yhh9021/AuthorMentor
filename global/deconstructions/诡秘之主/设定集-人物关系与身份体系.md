# 设定集-人物关系与身份体系：《诡秘之主.txt》

## 文件用途

本文件来自子 Agent 精读返工结果。它用于约束人物身份、关系变化和章节生成时的人物调用。

## 设定条目

## 1. 塔罗会是秘密组织的产品化设计

**设定定义**：塔罗会以愚者为中心，成员用牌名匿名交换情报、材料、委托和庇护。

**运行规则**：成员定期聚会，遵守等价交换和身份保密；现实任务通过各自社会位置执行，灰雾负责安全隔离与跨域调度。

**剧情落点**：第7章至终章。证据摘要：第7章塔罗会成立，第221章资源物流上线，戴里克、佛尔思、埃姆林等加入后不断扩展接口。

**代价与限制**：愚者必须维持神秘权威，成员每次求助都会把现实风险推向组织边缘，秘密泄露会牵连全员。

**人物和势力接口**：奥黛丽连接贵族和心理炼金会，阿尔杰连接海上和风暴教会，戴里克连接神弃之地，佛尔思连接亚伯拉罕，埃姆林连接血族。

**阶段变化**：从三人信息会发展为跨大陆、跨历史、跨信仰的隐秘网络，终局承担守护沉睡愚者的职责。

**复用边界**：成员必须有独立欲望和行动能力，不能只作为主角信息终端。

**写作用途**：组织成员各自代表地图与资源接口，主角无需亲自到场也能推进多线。

## 2. 锚把神明拉回人类社会

**设定定义**：锚是信徒、记忆、身份和情感对高位存在的固定力量，帮助其抵抗神性和原初意志侵蚀。

**运行规则**：信仰越稳定，神明越能保持自我；回应祈祷、塑造尊名和维护教团能强化锚。

**剧情落点**：第 575-640、703-740、1206-1270、1350-1394 章。证据摘要：拜亚姆海神线、达尼兹传播愚者、白银城走出神弃之地和终局塔罗会守护都围绕锚展开。

**代价与限制**：信徒诉求会成为责任，敌人可攻击信仰网络，高位存在也会被集体认知反向塑形。

**人物和势力接口**：海神信徒、白银城、塔罗会、家人记忆、教会仪式和尊名系统共同构成锚点来源。

**阶段变化**：前期克莱恩靠个人记忆自保，中期承接海神信仰，后期白银城和塔罗会成为愚者沉睡后的稳定锚。

**复用边界**：锚点要通过具体信徒和共同经历呈现，空泛崇拜无法承载情感。

**写作用途**：把信仰从背景崇拜改写成心理与战斗资源，使神格竞争具有人情重量。

## 3. 地图层级对应认知层级

**设定定义**：廷根、贝克兰德、海上、南大陆、神弃之地、星界和源堡不是单纯换景，而是信息和敌人层级逐步抬升。

**运行规则**：每换一层地图，主角获得新身份、新资源接口和新历史问题，同时保留上一层未清债务。

**剧情落点**：全书结构。证据摘要：廷根卷以邓恩之死留下复仇，贝克兰德卷以雾霾留下王室罪证，海上卷带出愚者信仰，神弃之地兑现白银城。

**代价与限制**：地图越大，普通人伤亡越重，主角私人行动越受组织和神战牵制。

**人物和势力接口**：廷根接教会，贝克兰德接政治和阶级，海上接殖民与信仰，神弃之地接古代历史，源堡接旧日。

**阶段变化**：从一座城市的自杀案扩展到大陆战争和星空末日，尺度提升始终由具体案件承接。

**复用边界**：新地图不能切断旧情感线；每次离开都要带走债务、敌人或资源。

**写作用途**：地图升级能自然带出世界观升级，使长篇结构保持阶段性新鲜感。

## 4. 身份马甲是社会接口

**设定定义**：克莱恩、愚者、夏洛克、格尔曼、道恩等身份对应不同阶层、地图、能力姿态和关系网络。

**运行规则**：每个身份有固定说话方式、资源入口、风险边界和可见战绩，切换身份必须处理信息隔离。

**剧情落点**：全书；无面人后强化。证据摘要：夏洛克追查贝克兰德案件，格尔曼在海上立威，道恩进入贵族圈，愚者维持塔罗会神秘权威。

**代价与限制**：马甲越多，记忆和关系债越复杂；一旦关联暴露，会牵连家人、塔罗会和盟友。

**人物和势力接口**：克莱恩接家庭和值夜者，夏洛克接贝克兰德调查，格尔曼接海上悬赏，道恩接贵族社交，愚者接信仰和源堡。

**阶段变化**：前期身份是求生伪装，中期成为行动工具，后期马甲反过来为愚者提供人性锚点。

**复用边界**：每个马甲都要有独立代价和不可跨用的信息，不能只是换称呼。

**写作用途**：同一主角可进入多层社会，不必强行让所有人认识同一个名字。

## 5. 复活不是重置，而是债务延续

**设定定义**：克莱恩从坟墓复活确认源堡带来有限复生能力，但每次复活都会改变身份、处境和敌我认知。

**运行规则**：复活受次数、时间、身体状态和源堡联系限制，不能保护所有同伴，也不能撤销灾难后果。

**剧情落点**：第212章起贯穿。证据摘要：第212-213章克莱恩参加自己的葬礼后从坟墓醒来，并在邓恩墓前立下复仇誓言。

**代价与限制**：复活带来亲人永别、身份清零、精神负担和敌人觊觎；死去的队长不会随主角回来。

**人物和势力接口**：廷根坟墓、源堡、占卜家途径、家人葬礼和后续马甲体系共同支撑复活叙事。

**阶段变化**：第一次复活用于卷末转场，后续成为高位争夺中的底牌和限制，终局则由沉睡替代简单复活。

**复用边界**：复活必须少用且每次改变局面，否则死亡威胁会失效。

**写作用途**：复活能力保留冒险强度，又通过不可挽回的关系损失维持重量。
