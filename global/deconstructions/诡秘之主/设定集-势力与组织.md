# 设定集-势力与组织：《诡秘之主.txt》

## 文件用途

本文件来自子 Agent 精读返工结果。写章节前先查本文件，确认规则、代价、人物接口和复用边界。

## 设定条目

## 1. 塔罗会是秘密组织的产品化设计

**设定定义**：塔罗会以愚者为中心，成员用牌名匿名交换情报、材料、委托和庇护。

**运行规则**：成员定期聚会，遵守等价交换和身份保密；现实任务通过各自社会位置执行，灰雾负责安全隔离与跨域调度。

**剧情落点**：第7章至终章。证据摘要：第7章塔罗会成立，第221章资源物流上线，戴里克、佛尔思、埃姆林等加入后不断扩展接口。

**代价与限制**：愚者必须维持神秘权威，成员每次求助都会把现实风险推向组织边缘，秘密泄露会牵连全员。

**人物和势力接口**：奥黛丽连接贵族和心理炼金会，阿尔杰连接海上和风暴教会，戴里克连接神弃之地，佛尔思连接亚伯拉罕，埃姆林连接血族。

**阶段变化**：从三人信息会发展为跨大陆、跨历史、跨信仰的隐秘网络，终局承担守护沉睡愚者的职责。

**复用边界**：成员必须有独立欲望和行动能力，不能只作为主角信息终端。

**写作用途**：组织成员各自代表地图与资源接口，主角无需亲自到场也能推进多线。

## 2. 正神教会兼具庇护与垄断

**设定定义**：黑夜、风暴、蒸汽、太阳等正神教会既保护普通人免受超凡灾害，也垄断知识、配方和封印物。

**运行规则**：官方小队处理非凡案件，封锁消息，按危险等级申请支援；跨教会合作在城市灾难和神战面前启动。

**剧情落点**：全书；廷根卷初步展开。证据摘要：廷根值夜者小队、贝克兰德多教会行动、大雾霾后风暴驱散和终局各神协作均体现此结构。

**代价与限制**：保密带来信息不对称，内部叛徒和神灵博弈会让基层牺牲，普通人难以知道自己为何受害。

**人物和势力接口**：值夜者、代罚者、机械之心、红手套、圣堂和各级主教构成官方非凡网络。

**阶段变化**：早期是克莱恩的庇护和课堂，中期与王室、邪教、半神阴谋博弈，后期成为末日前人类防线的一部分。

**复用边界**：垄断不能写成纯反派；基层守护者的牺牲必须与高层博弈区分。

**写作用途**：官方组织能同时提供秩序感和压迫感，使主角既受益又保持警惕。

## 3. 秘密组织构成地下资源市场

**设定定义**：密修会、极光会、心理炼金会、玫瑰学派、摩斯苦修会等组织分占不同途径、信仰和资源网络。

**运行规则**：组织通过配方、材料、成员晋升和信条维系凝聚，彼此合作、渗透、追杀和交易。

**剧情落点**：全书。证据摘要：瑞尔比伯与密修会、A先生与极光会、奥黛丽与心理炼金会、莎伦马里奇与玫瑰学派均承担组织接口。

**代价与限制**：脱离正神教会保护后，成员更容易被邪神污染、上级牺牲或高序列当作材料。

**人物和势力接口**：塔罗会成员的现实身份把这些组织接入灰雾，敌对任务则把它们接入各卷主线。

**阶段变化**：前期密修会追安提哥努斯笔记，极光会引爆廷根和雾霾，中后期心理炼金会、玫瑰学派和亚伯拉罕线扩展社会层级。

**复用边界**：组织要有独立目标、资源和行动风格，不能只换名字做同类反派。

**写作用途**：多组织生态让每个线索都有利益归属，避免世界只围绕主角旋转。

## 4. 王权成神需要历史与制度配合

**设定定义**：乔治三世的黑皇帝仪式依赖陵寝、秩序、战争、人口牺牲和王室资源，而非个人闭关突破。

**运行规则**：黑皇帝路径要求将国家秩序和皇帝象征纳入仪式，秘密陵寝与公共政策共同服务晋升。

**剧情落点**：贝克兰德后半至第1160章。证据摘要：卡平线、大雾霾、黑皇帝陵寝和第1143-1160章成神仪式共同回收王室布局。

**代价与限制**：成神工程把民众死亡、殖民战争和城市灾难转化为统治者资源，失败则引发国家震荡。

**人物和势力接口**：王室、军方、魔女教派、教会调查、塔罗会证据和黑皇帝牌共同构成仪式攻防。

**阶段变化**：从卡平和雾霾中的隐约王室影子，发展到道恩社交调查，再到乔治三世成神失败。

**复用边界**：国家级仪式必须有财政、战争、舆论和秘密工程痕迹支撑。

**写作用途**：将政治权力和超凡晋升合并，反派野心具备制度质感。

## 5. 纪元时间线让神话成为历史推理题

**设定定义**：第二纪古神、第四纪帝国、罗塞尔时代、第五纪鲁恩和未来末日组成多层时间线。

**运行规则**：每个时代留下文物、日记、封印物、遗迹和教会改写，读者需要通过互相矛盾的资料重建真相。

**剧情落点**：全书。证据摘要：罗塞尔日记从第21章起持续提供前史，阿兹克记忆和白银城线补全古代，终章说明旧日和源堡。

**代价与限制**：历史知识被神灵、教会和污染遮蔽，追索真相会触发高位注意。

**人物和势力接口**：罗塞尔日记、阿兹克记忆、格罗塞尔游记、白银城传说、黑夜女神说明和源堡真相互为证词。

**阶段变化**：早期历史只是背景，贝克兰德后变成案件线索，终局揭示穿越、源堡和末日都嵌在同一时间线。

**复用边界**：历史反转要保留前文证据，不能靠权威角色一次性讲完。

**写作用途**：长篇可以让历史碎片逐步校正读者认知，形成持续解谜感。
