# 设定集-写作复用边界：《高武纪元.txt》

## 可复用结构

- **公开弱项与私密强项双轨驱动**：换书时可把公开弱项改成血脉低、职业不被承认或魔法亲和差；私密强项必须具备可成长、可隐藏、可验证三点。
- **资源缺口先行，奖励兑现后置**：每个阶段列一个具体短缺项，让人物通过比赛、战功、合约或交易获得对应资源，并展示资源到账后的能力变化。
- **导师接力而非单师门包办**：为导师分配唯一职责：基础训练、制度推荐、世界观解释、危机护道、传承托付、道路选择；不要让同一人重复承担所有功能。
- **同代竞争梯队不断升级地图**：每个地图至少设置一个同阶标杆和一个超规格敌人，同阶负责比较，超规格负责逼出底牌。
- **战场收益绑定道德和死亡风险**：设计杀敌收益时，同步写出后勤、牺牲者、误判和敌方反制，让收益产生伦理厚度。
- **世界观由强者问答分层揭露**：把设定问题嵌入剧情需求：为什么要抢宝、为什么要保密、为什么敌人强、为什么道路难。
- **代号与化名制造多层声望**：为每个代号设计不同传播圈：战场代号、学院化名、神域身份、组织称号，并安排至少一次合流。
- **传承给宝物，也给义务**：设计传承时写三件事：传承者为什么留下、守护者付出什么、主角接受后必须做什么。
- **阶段胜利后安排私人回落**：每次大胜后安排一场低烈度关系戏，展示胜利影响了谁、主角在乎谁。
- **敌方理性追杀增强主角价值**：给敌方会议、成本估算或替代方案，让追杀显得是组织决策。
- **长篇地图以接口而不是瞬移扩张**：扩地图前先交代入口、推荐人、任务目标和失败代价，再让主角进入。
- **高阶权柄必须有规则限制**：给高阶权柄设置地域、对象、次数、代价或授权限制，并让主角用策略绕开。
- **时间跳跃服务阶段沉淀**：每次时间跳跃都配套一个“前后对比清单”：境界、身份、关系、敌人和新目标。
- **终局目标采用阶段闭环加开放牵引**：阶段末尾同时写“已完成什么”和“下一步为何更难”，并给出新目标名词和具体敌人。

## 不可复用边界

- **未觉醒者用神宫和苦功撬开高武起点**：新书改写时必须让私密能力有使用成本和信息遮蔽，若主角一开场就能公开碾压，贫寒、奖学金和考试压力会失效。
- **许博把学生推上资源轨道**：若导师直接替主角解决所有经济与竞争问题，成长爽感会从“争取资源”滑向“被动领奖”。
- **受法律保护的赌注引出战网与武殿合约**：赌约必须受规则约束并有第三方验证，否则容易变成无意义打脸；组织奖励也要和风险绑定。
- **二级战争警戒把少年目标压成文明责任**：亲人危机不能只当催泪按钮，必须改变主角的能力目标和行动选择。
- **月魔星界首次把杀戮变成成长反馈**：杀怪收益必须和真实战损并存；若只有收益没有死亡率、后勤和文明伦理，战场会退化成刷副本。
- **昆仑S级特训把校园天才改造成国家项目**：学院线要持续给任务和评估，不能只是换地图后的豪华宿舍。
- **百髓灵枯和十校战把身体痛苦写成路线选择**：痛苦蜕变要有明确后遗限制和人物照护，否则会成为廉价变强仪式。
- **鱼灵之潮让城市防线和武者天职合一**：故乡危机要推动人物关系和能力规则变化，不能只用城市受灾制造场面。
- **明墟飞蛇谷把夺宝写成上位者燃命护道**：燃命桥段不能频繁使用；每次燃烧生命都要换来关系升级或世界认知升级。
- **蓝星大赛和暗影刺杀把天才名声变成阳谋诱饵**：大赛后的追杀要有敌方理性计算，不能只因为反派嫉妒。
- **界中界用神明遗产把禁忌、传承和人性选择打包**：传承不能只给功法和宝物，还要给关系义务；否则神明遗产会变成无主礼包。
- **罗布海匿名灵枪把英雄荣誉藏进文明需要**：匿名英雄必须有现实理由，如保密、敌方追杀或组织安排，不能为了神秘而神秘。
- **天蛇神地揭开凡俗域与神域的大地图**：伙伴不能只当设定讲解员，必须在战斗、情感和资源线都有独立作用。
- **撕裂灵魂与半神战把主角逼进文明决策层**：精神突破不能只加数值，要改变视野、权限和敌人等级。
- **东方极之死把最强护道人改写成精神遗产**：假死或回归必须提前埋下信息差，否则会削弱死亡带来的情绪重量。
- **觉星大陆用化名许源完成神域试炼**：化名不能当独立人物处理；关系网要标注它是主角阶段身份，否则人物图会误判。
- **重归七星后的万魔覆灭把个人私心公开化**：私心要和公共利益发生真实摩擦，若只是给亲人奖励，无法形成角色厚度。
- **神庭总部抢徒把天才价值推到神王桌面**：招揽场景要服务修炼路线分歧，若所有强者只夸主角，容易变成空洞吹捧。
- **仙灵体系与源种把跨体系改写成价值审判**：如果异体系只是给主角刷资源，读者会觉得世界观膨胀但主题停滞。
- **北辰居把天神阶段写成神王棋局**：高阶战斗若缺少规则限制和外部势力，容易沦为数值互砍。
- **内域与帝兵弑渊把终局目标改成第一神王之路**：开放终点要给出当前阶段的实绩闭环，如斩敌、震慑和新目标，否则会显得只是断章。
- **七星历高武教育双轨**：教育设定要持续外溢到组织和战场，不能停留在校园排名。
- **心灵神宫与《观大日星空经》**：外挂要分层开放，早期只给辅助，后期才揭示因果；一次性全解释会损失追读动力。
- **灵性养料与觉醒阈值**：必须给上限和指引门槛，若无上限，主角会无限刷怪破坏剧情平衡。
- **身体素质、精神力与技艺三线战力**：数值要能在剧情里被验证，不能只堆面板数字。
- **武道资源经济**：资源不能只有名字，要写来源、价格、资格和争夺后果。
- **星火武殿合约与成员义务**：合约要有权利和义务双面，单纯高薪会削弱组织严肃感。
- **虚拟战网、段位和公开排名**：排名必须有后续代价，如被研究、被招揽、被追杀，才能避免变成孤立数值。
- **星界与前进基地**：每个星界要有差异化功能，不能只换怪物名字。
- **源力、源武者与飞天级**：突破名词要伴随能力变化，如飞行、源力循环、星术资格，不能只换称号。
- **星脉、生命星与功法雏形**：星脉数量不能随意通胀，要配合稀缺资源和不可逆后果。
- **星术、神晶和至强星术**：星术要受星脉和材料限制，不能学一本会一本。
- **法则真意与大道方向**：法则名词要通过具体战斗效果呈现，抽象讲道太多会拖慢节奏。
- **古神宫、万象星辰图与神明传承**：守护者要有独立意志和代价，不能只负责发任务。
- **遗迹禁忌与规则代价**：禁忌必须被执行，若警告后无人付出代价，设定会失去威慑力。
- **文明等级与敌对势力链**：每个敌方文明要有不同利益点，如宝藏、通道、斩首、神明布局，不能只写侵略。
- **凡俗域、神域与次元长河**：神域信息要分批释放，避免在中段一次性压垮主线。
- **神明等级与寿命阶梯**：高阶寿命不能让角色失去紧迫感，要用劫难、战争、道路瓶颈和因果压缩时间。
- **天蛇殿、圣血天蛇与柳冰契约**：伙伴要保留自主成长，不能只当主角装备。
- **祖界、祖界令与紫衣成员任务**：祖界令这类信物要有公开暴露机制，否则争夺缺少群体压力。
- **心梦神界、三帝宫与神庭权限**：权限要和能力、资源、责任同步提升，不能只作为头衔。
- **仙灵体系、伪天道与万神殿**：异体系要有自洽等级和社会秩序，不能只把修仙词汇套到高武数值上。
- **源种、《宇宙神体》与完美十星脉**：越强至宝越要有意识、代价或路线冲突，否则会破坏成长可信度。
- **北辰星、北辰居与内域五城**：每一层要有不同规则和目的，表层猎杀、内域炼化、五城权柄不能混同。
- **神国、神庭与帝宫首领身份**：身份要带来新限制和新敌人，不能只当称号奖励。
- **辟道、永恒道与神王道路**：道路要和主角早期性格、武器和选择呼应，临时拼贴会失真。
- **深渊主宰、圣祖与神帝级禁区**：神帝级名词要谨慎投放，必须有角色视角和行动后果承接。
- **公开弱项与私密强项双轨驱动**：若私密强项没有成本，外部弱项会显得虚假；若外界长期不更新评价，读者会觉得角色迟钝。
- **资源缺口先行，奖励兑现后置**：若资源缺口没有提前写，奖励会像作者硬塞；若奖励太多，贫困和竞争线会被冲淡。
- **导师接力而非单师门包办**：导师太多会稀释记忆点，需要让每位导师对应一个关键事件或代价。
- **同代竞争梯队不断升级地图**：若对手只负责被打脸，竞争梯队会失去说服力；必须让对手也有目标、情报或后续作用。
- **战场收益绑定道德和死亡风险**：只写收益会像游戏刷级，只写惨烈又会压住爽点；两者比例要按阶段调整。
- **世界观由强者问答分层揭露**：若问答没有当场影响主角选择，会变成百科插页。
- **代号与化名制造多层声望**：化名过多会混淆人物图，必须在文档中标注“许源是李源阶段身份”。
- **传承给宝物，也给义务**：只写传承奖励不写责任，会让高阶遗产像无人认领的仓库。
- **阶段胜利后安排私人回落**：私人回落不能过长拖慢主线，也不能只发福利，最好同时埋下下阶段责任。
- **敌方理性追杀增强主角价值**：若敌人多次低智送人头，主角威胁值会贬值；若敌人过强又无破绽，剧情会卡死。
- **长篇地图以接口而不是瞬移扩张**：只靠作者旁白说“更大世界”会显得地图膨胀；接口不足会让旧线断裂。
- **高阶权柄必须有规则限制**：没有限制会导致后期战斗失控；限制若太硬又会让升级不爽，需要保留关键时刻的压倒性兑现。
- **时间跳跃服务阶段沉淀**：跳跃过频会冲淡情感连续性，需要用故人、师门和旧承诺锚定时间。
- **终局目标采用阶段闭环加开放牵引**：若只开放不闭环会显得拖；若完全闭死又削弱追读。
