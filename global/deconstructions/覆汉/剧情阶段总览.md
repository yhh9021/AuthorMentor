# 剧情阶段总览：《覆汉.txt》

## 生成口径

本文件只保留全书粗粒度剧情阶段，目标是帮助写作系统快速理解主线推进。细颗粒事件因果见 `关键事件链.md`，设定年份顺序见 `设定集-设定时间线.md`。

## 全书阶段

### 1. 第 1-2 章 至 第 10-14 章：穿越母亲把历史知识变成家族经营 -> 请赏失败逼出大丈夫不可一日无权

**阶段主线**：现代女写手坠入锁龙井后成为辽西公孙氏寡妇，靠商贸、赈济、识字开蒙和边郡产业抚养遗腹子公孙珣。公孙珣起初把母亲关于黄巾、官渡、五虎将的说法当成离奇故事，直到公孙瓒被侯太守招为女婿、又被送往卢植门下，才意识到母亲的历史记忆正在兑现。 卢龙大捷后公孙珣替公孙越、韩当、程普请功。侯太守愿意给名族子弟公孙越打开户曹吏之路，却因韩当寒家出身拒给屯长，只愿多赏钱帛。公孙珣在家中雪夜醒悟权力决定功劳分配，随后亲自给阵亡士卒家送柴炭，韩当因此在封大河旁持刀投效。

**核心矛盾**：主角知道乱世将至，却还只是两百石主计副史；母亲要他借名师、同学和商号自保，他本人则既想求稳又被权力与名将刺激。 / 真正的敌人不是两千鲜卑骑兵本身，而是军中层级、上官懦弱、白身建言无效、夜袭失败后全员担责的制度压力。 / 军功与出身发生冲突：战场上能以勇力救人，官场上仍被太守一句话压回寒门。

**阶段兑现**：公孙珣在卢龙塞认出韩当，在旅途中盯住刘备、太平道、宦官子弟等节点，读者看到母亲的剧透不再是答案，而是引导主角做下注。 / 公孙珣用自身性命把假军令逼成真军令，韩当的勇力、公孙越的后方压阵、程普的主忧臣死同时兑现，战场上又把鲜卑内部商贸关系变成心理战武器。 / 公孙珣得到比战功更重的认知升级：掌权才能定义功劳；韩当看到公孙家对死者家属的照料，放弃塞障尉死路，成为主从关系的第一块基石。

**后续影响**：这条线把早期热血转成政治野心，后续招揽关羽、王修、娄圭时，都延续了不只看出身而看功能与心志的用人原则。

**关键事件入口**：第 1-2 章 穿越母亲把历史知识变成家族经营；第 2-9 章 三十骑夜袭卢龙塞外大营；第 10-14 章 请赏失败逼出大丈夫不可一日无权

### 2. 第 16-23 章 至 第 43-48 章、第 107-108 章：贾超兄弟案把太平道和宦官爪牙一起摆上台面 -> 纸张、伪书与石经让知识成为政治武器

**阶段主线**：公孙珣离开车队存问风俗，误入冀州乡里，卷入贾超杀豪强案。乡里里长、太平道人和大部分住户为贾氏兄弟遮掩，县令崔敏又因案子牵涉宦官子弟而两头畏惧。公孙珣借张俭、阳球的名望框架保人，也第一次近距离看到太平道的基层动员。 公孙珣利用软纸、印刷和文书便利经营义舍舆论与洛阳信息；蔡邕修石经、刘宽门生校经、卢植古文派回京，又把经学版本之争推到朝堂。公孙珣一边享受纸张带来的传播效率，一边被今文古文、师门名分、尚书台行事卷入政治。

**核心矛盾**：豪强攀附宦官，县令怕破家灭门；太平道能安抚整里百姓，却也在挑选血性勇夫；公孙珣若只守国法，会丢掉义气和观察乱世的入口。 / 义舍侵夺亭舍功能，收留逃犯与寒士，既会招官府忌惮，也会被卢植这样的师长一眼看穿动机。 / 知识不是中立技术：纸能传播规章和伪书，也能激化师门、学派、官学资格；主角若站错师门，会损害出仕路径。

**阶段兑现**：贾超脱离死局，韩当看见主君救人风格，公孙珣确认太平道不是神怪，而是扎进乡里、能使一百七八十户听令的组织。 / 公孙珣不靠空喊仁义，而用食宿制度筛出有故事、有本事、有怨气的人；卢植木碗压场让他意识到师生名分的威力，吕范随后成为文吏心腹。 / 公孙珣明白一张纸、一部经、一份书信都能改变政治博弈，后续伪造太平道书信、处理中台文书时都有前置训练。

**后续影响**：全书的文斗不靠玄学谋略，而靠文本载体、学派资格、官署流程和名士舆论协同推进。

**关键事件入口**：第 16-23 章 贾超兄弟案把太平道和宦官爪牙一起摆上台面；第 34-38 章 緱氏义舍把善名做成开放式人脉机器；第 43-48 章、第 107-108 章 纸张、伪书与石经让知识成为政治武器

### 3. 第 58-80 章 至 第 107-122 章：赵苞家眷危机把边郡救援升级为姻亲与军政秩序 -> 洛阳诛宦从私人愤怒变成中枢缝隙操作

**阶段主线**：鲜卑万骑入寇，辽西太守赵苞母亲和妻女被劫持，阳乐、柳城、管子城同时承压。娄圭提出冒险方略，公孙珣从卢龙出兵，在鲜卑、高句丽、乌桓、汉军和本地城池之间穿插救援，程普、韩当、公孙越各司其职。危机过后，赵家把赵芸许配给公孙珣，董卓又在边郡武人圈中登场赠刀。 公孙珣入中台后，从刘宽处得知王甫、曹节、段熲、田晏、夏育等人的利益纠葛。他在宫前与段熲对峙，借灾异与士人舆论推动诛宦气势，最后卷入王甫崩坏和乱箭杀局。

**核心矛盾**：救援对象是太守母亲，失败会震动洛阳；但边郡路远、情报混杂、部下互疑，娄圭的险计还触犯随行武人的安全感。 / 边军打胜仗要承受路线、补给、胡骑反扑和朝廷问责；同时洛阳中枢用宦官、郎官、尚书台重新定义这份战功。 / 公孙珣有诛宦意愿，却必须面对曹节尚能执政、王甫根基松动、段熲仍有军威、士人又想借刀的复杂局面。

**阶段兑现**：公孙珣在众怒中力保娄圭，靠分兵、奇袭与心理战化解边危，并将军功转成婚姻、名望和董卓等边将的认可。 / 公孙珣不仅在塞外击敌，也在郎署前与何进建立早期交情，在尚书台面对曹节、杨赐、袁逢等中枢人物。 / 宫前抽刀和地震灾异把私人胆气放大为政治符号，田丰、王允等士人围拢，王甫集团的孤立终于被利用。

**后续影响**：这段让公孙珣获得士林声望，也让他明白自己会被不同阵营当成刀使用，后续政治选择不再纯粹。

**关键事件入口**：第 58-80 章 赵苞家眷危机把边郡救援升级为姻亲与军政秩序；第 81-106 章 火烧弹汗把边将名声打进朝廷中枢；第 107-122 章 洛阳诛宦从私人愤怒变成中枢缝隙操作

### 4. 第 181-211 章 至 第 241-260 章：清理赵国太平道节点验证治理术 -> 阳翟群英会把同代英雄写成镜子阵

**阶段主线**：公孙珣任赵国地方官后修圪芦河、整顿豪强、使用王修执法，又对太平道采用分化策略：礼送一支、惩戒一支、放任一支，以伪造书信让钜鹿总部在马老公和张晟之间作取舍。张晟愤而质问士人不顾野草般百姓，公孙珣既承认太平道的民间价值，又把它从本地治理中剥离。 黄巾战局中，曹操暑日怀霜雪、傅燮乘夜拭白珪、孙坚所向无前、公孙珣且战且思、刘备作歌、钟繇荐英雄等段落接连出现。每个人都在同一乱局下显露不同品性：曹操清醒冷峻，傅燮洁烈，孙坚勇猛，刘备能得人，公孙珣擅把战事与政治收益同算。

**核心矛盾**：太平道是信仰、互助、豪强渠道和潜在叛乱网络的混合体；粗暴镇压会逼乱，纵容会破坏水利和政令。 / 黄巾不是一城一地的乱贼，而是全国性组织；朝廷旧军和地方官既要镇压，又要防止河北彻底崩坏。 / 同一目标下，各英雄的价值观、出身、组织资源和风险承受能力不同，合作里天然带着未来分路。

**阶段兑现**：娄圭伪书计与公孙珣的分化处置让太平道本地节点自解，王修、董昭等行政骨干看到主君既能用法，也能用势。 / 公孙珣借誓盟和三策稳定部众，轻兵双城、铁骑横百里让读者看到治理官员切换为战时统帅的能力。 / 读者同时获得历史熟人登场的兴奋和重新评估的乐趣，公孙珣也在比较中确定自身路线。

**后续影响**：这组群像为后期曹操、刘备、孙坚、傅燮各自成势或殉节建立情感底色，旧友成敌时更有重量。

**关键事件入口**：第 181-211 章 清理赵国太平道节点验证治理术；第 218-240 章 岁在甲子前夜把天下叛乱写成早已埋好的账；第 241-260 章 阳翟群英会把同代英雄写成镜子阵

### 5. 第 301-320 章 至 第 339-380 章：节使三河与西凉战线让朝廷军权露出旧病 -> 关东起群雄后旧友旧将被迫分线

**阶段主线**：黄巾后，公孙珣节使三河募年少，面对韩遂、西凉、关中和朝廷名将体系。卫青李广典故、白马猎长原、叛军崩殂等段落，把主角从河北治理者推向更大的军事调度，同时呈现边郡人与关东士人长期互不信任。 关东群雄兴起，公孙珣拆分旧将左右军，面对董卓、袁绍、韩遂、吕布、贾诩等人各自占位。平城候骑、单于在山西、降虏西击胡、大河封冻乱武兴等事件，把讨董前后的军事、边胡和政治选择纠缠在一起。

**核心矛盾**：朝廷要用边将平乱，又害怕边将坐大；关东士人看不起西凉，西凉人物反过来认为中枢尸位素餐。 / 辽东不是单纯战场，而是汉吏、边民、胡部、贸易下线和高句丽势力交错的区域；武力太弱镇不住，武力太重又损害长期治理。 / 旧交、旧恩、同为边将的互相欣赏，在天下崩坏后不足以压过地盘、天子、兵权和路线差异。

**阶段兑现**：公孙珣以三河兵、白马骑和旧边将经验处理叛军，既得军功，也更清楚朝廷无法稳定整合地方武力。 / 公孙珣通过快速压服与文书收编，让边地从敌我混杂转成可征发、可贸易、可驻军的后方。 / 公孙珣不再以单点胜利解决问题，而是通过分军、候骑、降虏、借胡制胡和政治表态同时推进。

**后续影响**：这段把全书从朝廷仍可修补的时代推入诸侯时代，人物关系开始大规模由同盟转为阶段性敌友。

**关键事件入口**：第 301-320 章 节使三河与西凉战线让朝廷军权露出旧病；第 321-338 章 辽东半月平定把伐心写成强权治理；第 339-380 章 关东起群雄后旧友旧将被迫分线

### 6. 第 400-440 章 至 第 481-518 章：河北袁绍大战把名士领袖逼成战略主敌 -> 曹操旧友成中原对手，沂水斩龙收束英雄时代

**阶段主线**：袁绍与公孙珣在河北进入正面争夺。梁期孤城、河东猛虎、关塞行路、旧渡借归船、漳河连清河、伏盾终起盾、三面成围等段落，写的不只是战役胜负，而是名士领袖与北方建制者争夺河北秩序的决战。 曹操由早年洛阳互赏旧友，发展为中原联军核心。紫气黄旗之后，公孙珣与曹操、刘备、袁术、刘表、孙氏等势力在南北格局中重排位置。沂水、断龙背、一朝斩龙等段落把军事胜负、旧友离别和天下定向合在一起。

**核心矛盾**：袁绍有士族号召力和名分，公孙珣有边军、治理、白马旗和北方根基；双方都能吸纳人才，也都要面对内部文武旧患。 / 汉室仍有礼法惯性，旧臣仍讲忠义；公孙珣的军政实力若没有叙事和程序承接，就会被看成篡夺。 / 情义不能取消战略矛盾：曹操要维持中原系统，公孙珣要完成北方与天下新秩序，两者都不能退让。

**阶段兑现**：战局通过孤城、请降、解衣见血、伏盾、围城层层推进，最后把袁绍从旧友圈人物推成河北路线的最大障碍。 / 光武、赤伏、云台等典故让新政权不是突然加冕，而是被置入汉家循环、功臣秩序和天下安定的论证中。 / 战役不靠口号分胜负，而用联舸、沂水、虎穴、断龙背等连续军事场景完成旧友对决。

**后续影响**：曹操线把全书从河北内战推向天下收束，也让刘备南行、旧友南北分路更显苍凉。

**关键事件入口**：第 400-440 章 河北袁绍大战把名士领袖逼成战略主敌；第 441-482 章 赤伏与光武叙事把新政权合法性前置讨论；第 481-518 章 曹操旧友成中原对手，沂水斩龙收束英雄时代

### 7. 第 519-541 章：故垒萧萧与铜雀迟来让胜利落到人世沧桑

**阶段主线**：终局阶段不再只写统一后的庆功，而连续写大雪、剑阁、铜雀、枯枝培土、故垒萧萧、故人送客和天若有情。公孙珣建立新秩序后，旧友、旧敌、家族继承和功臣去留都要被安放，最后以人间正道是沧桑收束。

**核心矛盾**：打下天下不等于结束故事；功臣、旧汉臣、亡故者、南行旧友和下一代都在要求主角给出解释。

**阶段兑现**：终章把宏大历史降回人情：不是单纯登极，而是承认旧时代死亡、新时代也要背负旧账。

**后续影响**：全书的历史感因此完整闭环，读者记住的不只是谁赢了，还有胜利造成的离散、牺牲和制度沉淀。

**关键事件入口**：第 519-541 章 故垒萧萧与铜雀迟来让胜利落到人世沧桑
