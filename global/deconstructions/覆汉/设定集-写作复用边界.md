# 设定集-写作复用边界：《覆汉.txt》

## 可复用结构

- **史实预期差驱动**：选一个读者有共同知识的历史或行业节点，把大事件拆成主角早期能接触到的小人物、小制度和小误差。
- **制度门槛制造升级感**：为每个阶段设置一个清晰门槛：资格证、师承、许可证、官位、董事会席位等，再让胜利转换成进入下一门槛的凭据。
- **小事件预演大时代**：每个宏大危机先设计一个微观样本，让主角亲眼进入并付出选择成本。
- **人脉不是收藏，而是功能分工**：招揽人物前先确定空缺功能，再用事件证明此人填上空缺。
- **战后分配比战斗更能塑造关系**：每场重要胜利后安排一场结算戏，写谁得官、谁得钱、谁背锅、谁受委屈。
- **公共设施招人法**：在新书里设计医馆、驿站、开源社区、难民营、训练馆等开放节点，配套规则和管理者。
- **关系阶段反转**：给每条核心关系写早期善缘、中期利益分叉、后期对立场景，不用单标签锁死人物。
- **治理成果转军事实力**：安排一项民生工程，并让它在后续战役中贡献粮道、桥梁、民夫、情报或合法性。
- **伪史书引文制造终局回声**：在关键节点插入短小、风格统一的后世档案、判词、史评或新闻回顾，但只给信息增压，不替代正文冲突。
- **合法暴力先拿大义**：动手前安排一场合法性铺垫：敌人失德证据、上级授权、公众舆论、旧法条、仪式。
- **地图节点推动剧情换挡**：为长篇每个大地图定义一句功能：学校、战场、官署、粮仓、边关、都城、决战地。
- **胜利后清点代价**：最终大战后安排一组回访：旧战场、旧师友、旧敌家属、下一代、制度遗产。

## 不可复用边界

- **穿越母亲把历史知识变成家族经营**：不能让未来知识直接替代行动；每次预判都要附带社会成本、信息偏差和本土角色的反作用。
- **三十骑夜袭卢龙塞外大营**：复用时要有明确军事逻辑和后勤逻辑，不能只写少数人冲营就让大军崩溃。
- **请赏失败逼出大丈夫不可一日无权**：不能把挫败写成单纯压主角；挫败必须导出新目标、新关系或新制度理解。
- **贾超兄弟案把太平道和宦官爪牙一起摆上台面**：案件不能只当悬疑桥段，必须让它改变主角对世界运行规则的认识。
- **緱氏义舍把善名做成开放式人脉机器**：不能只写主角大方撒钱；要把住宿、饭食、登记、惩戒、告示等运转细节写出来。
- **纸张、伪书与石经让知识成为政治武器**：不能让技术一出场就碾压时代；必须写出旧制度吸收、限制和反咬新工具的过程。
- **赵苞家眷危机把边郡救援升级为姻亲与军政秩序**：救援线不能只靠主角神勇，必须让地图距离、敌军动机、城池位置和援军身份都参与计算。
- **火烧弹汗把边将名声打进朝廷中枢**：不能只给主角战后升官；要写出朝廷如何询问、审核、利用、消化这场胜利。
- **洛阳诛宦从私人愤怒变成中枢缝隙操作**：反派不能只是坏；要给其派系、爪牙、旧功、贪腐和内部敌人，主角的胜利才有政治可信度。
- **清理赵国太平道节点验证治理术**：不能把民间组织写成纯恶；要保留它给贫民提供依靠的一面，主角处置才有道德重量。
- **岁在甲子前夜把天下叛乱写成早已埋好的账**：复用时不能突然宣布天下大乱；要先写乡村、县令、士人、豪强、宗教的连续破口。
- **阳翟群英会把同代英雄写成镜子阵**：群像不能只靠名号堆砌；每个角色都要承担独立抉择和不同后果。
- **节使三河与西凉战线让朝廷军权露出旧病**：不能把边地军阀写成单一野心家；他们要有被中枢利用、轻视和反噬的历史债。
- **辽东半月平定把伐心写成强权治理**：复用时不要把异族和边民当背景板；他们要有自己的粮食、语言、交易和向上通道。
- **关东起群雄后旧友旧将被迫分线**：不能让阵营变化突兀；每条分线都要有早期交情、资源矛盾和转折事件。
- **河北袁绍大战把名士领袖逼成战略主敌**：主敌不能靠降智失败；要让其优势真实存在，主角胜利才有价值。
- **赤伏与光武叙事把新政权合法性前置讨论**：不能把合法性写成一句天命所归；要让不同阶层各自得到能接受的理由。
- **曹操旧友成中原对手，沂水斩龙收束英雄时代**：不能为了终局强行黑化旧友；对手的理想、能力和不得不战的理由都要保留。
- **故垒萧萧与铜雀迟来让胜利落到人世沧桑**：结局不能只列年表；要让具体人物在新秩序中得到命运回声。
- **半土著先知规则**：复用时要让本土制度反向塑造主角，不能让现代记忆跳过地方官场、军功和士族规则。
- **经学即官场学历**：不能把学院写成现代学校；要写束脩、荐书、出身、师门名分和学派争斗。
- **州郡征辟与吏员秩级**：不要只写品级名词；要体现谁能任命、谁担责、谁受举主牵制。
- **首级军功与门第过滤**：不能让功劳自动公平兑现；必须写出功劳进入制度后的损耗。
- **安利号边郡贸易网络**：商号不能万能；必须明确它能影响小部族、商旅和家族，却不能直接支配朝廷和大军。
- **卢龙塞与幽州门户**：复用时不要只写一座关；要写它连接哪些郡、河流、商路和敌军行动。
- **胡汉战争与互市并存**：不能把异族写成统一意志；要区分王庭、大部落、小部族、汉化商贩和内附人群。
- **党锢后的士人舆论**：不能把士人写成单纯正义；他们同样维护推荐权和家族利益。
- **宦官权力的内外链条**：不能只写宫斗；要把宫中命令如何落到县里、军中、婚姻和豪强身上写清楚。
- **中台、郎署与黄门三角**：要写官署分工、门禁、文书和接待层级，不能把所有权力都压成皇帝一句话。
- **义舍、告示板与公共空间**：公共空间必须有规章和管理者，否则会变成主角善良的空洞展示。
- **太平道三十六方**：不能只写邪教蛊惑；要写百姓为何信、豪强为何投、官府为何失灵。
- **水利治理转军事资本**：不能只写开渠成功；要写劳役、反对者、信仰组织、上级考核和刺杀风险。
- **白马旗与私人军队建制**：旗号必须由连续战绩支撑，不能只靠命名制造威名。
- **婚姻与继承是政治接口**：不能把婚姻只写成奖励；要写家族、门第、继承、舆论和情感牵引。
- **会盟、歃血与诸侯联盟**：不能把会盟当同心同德；盟约下必须保留算计和退出成本。
- **天命、谶纬与新政权合法性**：天命不能凭空降临，要由战功、治理、旧典、舆论和继承安排共同支撑。
- **阶段时间线：熹平求学到终局建制**：不能让阶段只换地图；每次换阶段都要替换主角的权力接口、敌人形态和代价。
- **史实预期差驱动**：只照搬史实会像年表；只颠覆史实会失去期待。关键是每个偏差都要有本世界因果。
- **制度门槛制造升级感**：门槛写得太空会拖节奏；要用具体人物卡住主角，再用具体收益打开。
- **小事件预演大时代**：微观事件若与后续大事无关，会显得支线堆砌；每条小线都要在后期成为解释大势的证据。
- **人脉不是收藏，而是功能分工**：只按名气收人会造成脸谱和拥挤；缺少岗位的人会变成摆设。
- **战后分配比战斗更能塑造关系**：跳过分配会让组织成长失真；分配戏没有情感对象则会像账本。
- **公共设施招人法**：设施若没有规则，会变成善举背景；若没有风险，会缺乏戏剧张力。
- **关系阶段反转**：反转缺少利益理由会像背叛；只写利益又会丢情感厚度。
- **治理成果转军事实力**：治理线太平会显枯燥；要让豪强、宗教、上级和劳役冲突持续压迫它。
- **伪史书引文制造终局回声**：引文过多会打断叙事；引文若只重复正文会显炫技。
- **合法暴力先拿大义**：只讲大义不动手会虚；只动手无名义会像匪气。
- **地图节点推动剧情换挡**：地图只换地名会疲劳；必须换规则和人物接口。
- **胜利后清点代价**：只庆功会浅；只悲伤会削弱胜利。要让代价和新秩序并存。
