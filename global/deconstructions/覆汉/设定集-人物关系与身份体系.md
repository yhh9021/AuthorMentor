# 设定集-人物关系与身份体系：《覆汉.txt》

## 文件用途

本文件只收人物，不收规则、组织、地点或抽象设定。它用于区分主角、核心人物、重要人物和次要人物，并约束章节生成时的人物调用。

## 主角与核心人物

### 1. 卢植

**重要度**：主角/核心人物

**功能定位**：导师上位者/古文大儒

**身份弧线**：海内名儒兼能臣，从记名老师变成真正压住公孙珣士林身份的师长。

**关系网功能**：提供经学名分和严厉约束，代表古文派、师道和汉臣气节。

**关键关系**：
- 公孙珣 -> 卢植（师徒/政治背书）：记名弟子后成为受约束的门生。；变化：从远道求学，到义舍被师长压场，再到士林身份定型。
- 公孙瓒 -> 卢植（师徒/潜在裂痕）：公孙瓒名义上为卢植门下，但对卢氏子弟和严厉师门心存不满。；变化：从求学依附到后期与卢氏疏离。
- 卢植 -> 刘宽（师友/学派分歧）：二人私交深厚，却分别代表古文与今文重要资源。；变化：从酒友师长变成学派张力来源，再以君子私谊化解部分压力。
- 卢植 -> 蔡邕（文化合作/士林网络）：同属修石经与经学文化圈的重要名士。；变化：从文化名士往政治文本权力靠拢。

**章节范围与证据**：第 24-438 章。第 24 章緱氏山下，第 34 章义舍木碗压场，第 52 章参与定字。

**调用注意**：写到 卢植 时，必须同时检查其身份阶段、利益牵引、与主角关系的变化，以及其是否会改变当前事件的权力结构。

**关系网功能**：给主角入室师承和洛阳上层接口，也在诛宦前提醒中枢格局。

**关键关系**：
- 公孙珣 -> 刘宽（师徒/入室背书）：刘宽收公孙珣、公孙越为入室学生。；变化：从求名师背书变成洛阳内堂政治咨询。
- 公孙瓒 -> 刘宽（师徒/上升通道）：公孙瓒因仪表和机缘被刘宽收为学生。；变化：偶然借车变成公孙兄弟入室师承。
- 卢植 -> 刘宽（师友/学派分歧）：二人私交深厚，却分别代表古文与今文重要资源。；变化：从酒友师长变成学派张力来源，再以君子私谊化解部分压力。

**章节范围与证据**：第 29-387 章。第 29 章收徒，第 107 章内堂谈王甫、曹节与何进。

### 3. 曹操

**重要度**：核心人物

**功能定位**：早期友人/后期战略对手

**身份弧线**：以杖杀宦官亲族、顿丘令、洛阳清醒人物形象进入，后成中原系统核心。

**关系网功能**：提供英雄互赏和终局对照，旧友变敌提高情感重量。

**关键关系**：

**章节范围与证据**：第 123-540 章。第 123-145 章互赏交游，第 244 章曹孟德暑日怀霜雪，第 481-500 章中原对峙。

- 公孙大娘 -> 公孙瓒（家族扶持/史实下注）：公孙大娘早年接济公孙瓒，并让公孙珣借他求学和人脉。；变化：从家族接济变成主角求学入口。
