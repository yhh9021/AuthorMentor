# 设定集-资源体系：《覆汉.txt》

## 文件用途

本文件由设定体系专题子 Agent 独立生成。写章节前用它校验规则、代价、资源接口和禁忌边界。

## 设定条目

## 1. 安利号商业网络

**类型**：资源体系

**设定定义**：以辽西公孙氏和公孙大娘经营的商号为基础，连接东北、河北、青州和塞外的财货网络。

**运行规则**：商路提供钱、马、粮、布帛、铁器、货栈和下线部族；它能支付求学、聘礼、抚恤、义舍和军事前置成本。

**剧情落点**：第2章、第5章、第58章。证据摘要：第2章写商号经营近二十年，分号从乐浪到邺城，并经手马、粮、丝帛和铁器；第5章夜袭前以商号承担士卒家属抚养；第58章娄圭指出其货栈、商队和塞外下线可支撑军事设想。

**代价与限制**：商业网络依赖边郡通道和家族信任；战乱会把商路收益转为被征调、被觊觎和被断路的风险。

**人物和势力接口**：卢龙塞商路、辽东大马、河北粮帛、青州铁器、塞外部族下线。

**阶段变化**：第2章作为家族财富来源登场；第5章直接支付夜袭抚恤；第58章被娄圭看作乱世占塞和控制交通的基础。

**复用边界**：不能只写有钱；要写具体货品、节点和使用场景。

**写作用途**：让主角早期资源不凭空出现，财力和地缘绑定。

## 2. 师承与义舍人脉资源

**类型**：资源体系

**设定定义**：名儒师承提供士林入场券，义舍把散财和住宿规约变成持续吸纳寒门、亡命、商旅、士子的网络。

**运行规则**：师承负责上层认证，义舍负责低成本筛选和聚合；饭食、号牌、告示、禁斗与管事共同维持秩序。

**剧情落点**：第24-38章、第43-48章。证据摘要：第32章提出在緱氏建不分身份的义舍；第34章写告示板、住宿规约和管事；第35章卢植用师生名分压住公孙珣；第43-48章义舍网络延伸到纸张和经学争端。

**代价与限制**：义舍侵入亭舍和官府功能，收留复杂人群会招忌；师门名分也能反过来约束经营者。

**人物和势力接口**：卢植刘宽门下、緱氏义舍、告示板、寒门士子、亡命者。

**阶段变化**：第24-32章求学取得资格；第34-38章义舍正式运转；之后纸张、伪书、太平道和幕府招纳都复用这套网络逻辑。

**复用边界**：不能只写广施恩惠；必须写登记、规则、惩戒和被上位者审视。

**写作用途**：可借鉴为历史题材的开放式招募装置。

## 3. 兵粮财政与抚恤体系

**类型**：资源体系

**设定定义**：战争资源由府库、商号、民夫、粮草、马匹、甲仗和战后抚恤共同构成。

**运行规则**：能不能动兵，先看兵员编制、粮械供应、马匹状态和士卒后顾；抚恤承诺能把临时冒险转为可组织行动。

**剧情落点**：第5章、第230章、第344章、第380章。证据摘要：第5章列名录后承诺布帛钱粮和家属抚养；第230章说明北军五校和羽林虎贲人数有限，出兵受兵源限制；第344章各路关东军以府库和征兵扩张；第380章蒲津渡因渡河携械有限而付出高伤亡。

**代价与限制**：缺兵缺粮会拖延出击；远征会受渡河、军械和局部兵力限制；抚恤不到位会损坏军心。

**人物和势力接口**：北军五校、地方郡兵、商号钱粮、民夫陪隶、渡河军械。

**阶段变化**：第5章以抚恤启动夜袭；第230章黄巾爆发后中枢有将无兵；第344章诸侯各自搜刮府库扩军；第380章攻蒲津时物资不足影响战场。

**复用边界**：不能让大军凭空集结；要说明兵从何处来、粮械如何到、伤亡如何结算。

**写作用途**：让战争规模受财政和后勤约束，增强历史真实感。

## 4. 骑兵、边马与胡汉向导资源

**类型**：资源体系

**设定定义**：骑兵能力依赖边郡马源、乌桓突骑、熟悉塞外道路的部族向导和边军训练。

**运行规则**：骑兵资源提供速度差和侦察优势；出塞远袭必须配备识路者、上膘马匹、轻重骑配合和可接应的步军。

**剧情落点**：第3-8章、第90-96章、第235-237章。证据摘要：第3章韩当作为骑卒什长提出敌营虚实；第90章公孙珣向辽西要来熟悉弹汗山道路的莫户部勇士，汉军和乌桓突骑合计万骑；第235-237章轻兵取城后以骑兵追击扩大战果。

**代价与限制**：骑兵昂贵且补给脆弱；失去向导或误判地形会让速度优势变成孤军深入。

**人物和势力接口**：辽东大马、乌桓突骑、莫户部向导、边郡精锐、白马骑兵。

**阶段变化**：第3-8章小规模骑卒夜袭；第90章万骑出塞依赖乌桓突骑和向导；第236-237章铁骑横百里成为黄巾战场扩大战果的手段。

**复用边界**：不能只写骑兵快；要写马源、向导、季节、地形和步骑协同。

**写作用途**：把骑兵写成资源体系，而非单纯兵种标签。

## 5. 信息差与文本媒介资源

**类型**：资源体系

**设定定义**：未来记忆、商路消息、告示纸张、伪书印绶和中台文书共同构成信息资源。

**运行规则**：信息差提供方向，商路和斥候提供更新，纸张和文书提供传播，中枢流程提供执行；四者结合才能影响政治现实。

**剧情落点**：第1-2章、第43-48章、第108章、第211章。证据摘要：第2章历史记忆引导求学和商号经营；第43章伪书借印绶产生政治效力；第47章纸张相较书简便宜轻便但耐久受限；第108章尚书台和黄门承担中枢信息流；第211章假信拆解太平道节点后留下怨恨。

**代价与限制**：过期信息会误导判断；伪书和盗印触犯刑律；传播越快，失控也越快。

**人物和势力接口**：历史记忆、商队下线、义舍告示、软纸、尚书台文书。

**阶段变化**：第1-2章以母亲历史记忆和商路见识开局；第43-48章信息载体进入士林斗争；第108章信息处理转入中枢文书系统；第211章伪信用于治理分化并付出道德代价。

**复用边界**：不能把情报写成全知视角；每条信息都要有来源、时效和传递成本。

**写作用途**：适合构造非玄幻能力：信息优势必须靠媒介和组织落地。
