# 设定集-人物关系与身份体系：《鸣龙》

## 文件用途

本文件来自子 Agent 精读返工结果。它用于约束人物身份、关系变化和章节生成时的人物调用。

## 设定条目

## 1. 甲子莲、凤羽草与高阶药材

**设定定义**：甲子莲、凤羽草等药材是高阶救治和破局资源，往往绑定门派地盘、旧怨和强者承诺。

**运行规则**：药材不能随意购买，需以人情、战功、交易或闯遗迹取得；药材的归属会改变势力关系。

**剧情落点**：第148章；第166-183章；第221-224章。证据摘要：第148章“血妖丹”附近强化药丹与邪丹对照；第166-183章解毒救治；第221-224章火凤谷、朱陵火府承接凤羽草资源线。

**代价与限制**：求药者要承担欠债、涉险、被门派卷入和触发旧案的代价；药材本身也有用错后的反噬。

**人物和势力接口**：连接林婉仪医术、林紫苏丹法、南宫烨、步月华、丹王、火凤谷。

**阶段变化**：中期从单次救命需求升级为火凤谷和朱陵火府的地图入口，后续与封镇体系相连。

**复用边界**：资源线要绑定具体病人或旧债；脱离人物，药材名称会变成清单。

**写作用途**：用救人资源驱动地图扩张，比“主角想变强所以去秘境”更有情绪动力。

## 2. 赤麟卫、京兆府与大乾律

**设定定义**：赤麟卫和京兆府构成大乾处理妖案、权贵案件和京城治安的执行体系，大乾律则限制官员与江湖人的行动边界。

**运行规则**：妖案可由捕快、赤麟卫、钦天监和相关官府协同；京城案件还要受皇权脸面、卷宗程序和官员派系影响。

**剧情落点**：第5-6章；第83-143章。证据摘要：第5-6章丹阳盘查谢尽欢；第83章后京城旧案围绕赤麟卫失职、卷宗和权贵处分展开。

**代价与限制**：制度提供合法性，也制造拖延和遮掩；谢尽欢若越权，会从英雄变成嫌犯。

**人物和势力接口**：连接令狐青墨、杨大彪、韩靖川、李公浦、周明安、乾帝和谢温旧案。

**阶段变化**：丹阳阶段是地方执法压力，京城阶段变成谢尽欢借力复仇的制度工具。

**复用边界**：制度戏需要具体职位和流程；本书用捕快盘问、卷宗、失职、体面处置落地。

**写作用途**：让官府不是背景板，而是既阻碍又可利用的规则场。

## 3. 北周女武神与太后身份

**设定定义**：郭太后兼具北周太后、女武神、三年前救命恩人和谢尽欢情感对象身份，是政治权力与武道巅峰的复合角色。

**运行规则**：她的公开身份代表北周政权，武道身份代表天下顶层战力，私人身份则通过救命恩和绯闻与谢尽欢相连。

**剧情落点**：第235-287章；第621章。证据摘要：第235章“女武神”、第274章“太后有请”、第621章“天地同寿”分别推进身份、政治和终局代价。

**代价与限制**：任何接触都会被大乾、北周和江湖解读成政治信号；她在终局使用天地同寿时还承担自我消融风险。

**人物和势力接口**：连接谢温下落、金甲救命、凤仪司、红豆、北周朝局、天地同寿和大婚。

**阶段变化**：早期是传闻里的荒唐太后，中期露出救命恩和女武神，终局成为对抗尸祖的核心巨头。

**复用边界**：复合身份要分阶段揭开；若一开始全盘公开，谣言伏笔和政治张力都会消失。

**写作用途**：把强者、政权和情感集中在一人身上，能提高每次出场的信息密度。

## 4. 谢尽欢

**设定定义**：谢尽欢 的身份功能是“失忆开局的主角，谢温之子，丹阳英雄，后期持鸣龙参与镇压尸祖。”，主要承担 把案件、感情、封印、政治和武道主线全部串联起来。

**运行规则**：调用 谢尽欢 时，先确认其在 全书多阶段 的身份阶段，再围绕其关系功能安排行动、信息、压力或兑现。

**剧情落点**：全书多阶段。证据摘要：谢尽欢 在《鸣龙》人物关系网中承担“把案件、感情、封印、政治和武道主线全部串联起来。”功能。

**代价与限制**：复用 谢尽欢 的功能位时，必须改写姓名、阵营、利益和互动场景；若只保留标签而不保留“把案件、感情、封印、政治和武道主线全部串联起来。”，人物会失去剧情作用。

**人物和势力接口**：把案件、感情、封印、政治和武道主线全部串联起来。

**阶段变化**：从镇妖陵事故逃出的嫌疑人，成长为能在官场、江湖、宗派和终局战中借势破局的守界者。

**复用边界**：可借鉴 失忆开局的主角，谢温之子，丹阳英雄，后期持鸣龙参与镇压尸祖。 的结构作用，不能迁移 谢尽欢 的专名、原书标志性互动和原有人物绑定关系。
