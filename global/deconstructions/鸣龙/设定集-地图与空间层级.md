# 设定集-地图与空间层级：《鸣龙》

## 文件用途

本文件来自子 Agent 精读返工结果。写章节前先查本文件，确认规则、代价、人物接口和复用边界。

## 设定条目

## 1. 武道品级与登仙门槛

**设定定义**：武道以九品至高阶、再到超品和登仙层级展开，品级不仅代表力量，还影响是否能接触镇妖陵、冥寂、天地同寿等高级规则。

**运行规则**：低品靠招式和气血对抗，中高品可破煞、抗蛊、入阵，高阶强者能触及神魂、天地灵韵和世界级封印。

**剧情落点**：全书；重点第53-63章、第183章、第621-624章。证据摘要：丹阳赌船靠道行暴涨丹临时跨品；丹王赠鸣龙说明破煞价值；终局冥寂需要高阶武道和天地规则对抗。

**代价与限制**：越级对抗需要丹药、法器、地利或同伴协力；强行提升会带来失控、伤损和心性风险。

**人物和势力接口**：与道行暴涨丹、鸣龙枪、女武神、尸祖冥寂、宗派地位和朝廷威慑相连。

**阶段变化**：早期谢尽欢靠技巧和夜红殇辅助越级，中期以鸣龙枪和战斗经验稳定上升，终局必须联合高阶强者对抗尸祖。

**复用边界**：境界台阶要和具体战术配套；只喊品级会削弱战斗可信度。

**写作用途**：品级体系既给爽点台阶，也给失败边界，读者能理解主角何时能硬打、何时必须借势。

## 2. 八方通明阵与钦天监预警

**设定定义**：大乾用钦天监、阵法和宗派协防监控妖邪动向，八方通明阵代表国家层面的超凡预警能力。

**运行规则**：大型邪术、血煞、妖气会触发阵法或被钦天监推演；地方官府和赤麟卫再按线索追捕。

**剧情落点**：第83-143章；第189-216章；第487章后。证据摘要：丹阳伪造血煞能误导追查，京城与钦天监、赤麟卫关联紧密，天南血煞潮考验国家防线。

**代价与限制**：阵法只能发现异常，不能替人判断真伪；高手和邪教会用伪装、转移、内应和舆论扰乱判断。

**人物和势力接口**：连接赤麟卫、京兆府、宗派巡守、冥神教血祭、谢尽欢伪造血煞。

**阶段变化**：早期作为追查谢尽欢的威胁，中期变成朝廷与宗派博弈工具，后期面对血煞潮时暴露防线压力。

**复用边界**：预警系统不能全知；本书让它受到信息差和政治干扰限制。

**写作用途**：给国家安排超凡预警系统，能解释为什么邪教需要遮掩和分段推进。

## 3. 地图层级：丹阳到麒麟洞

**设定定义**：地图从徽山镇妖陵、丹阳、京城、三江口、火凤谷、北周、烟波城、天南、麒麟洞逐层扩张，每一层都对应更高的制度和灾害规模。

**运行规则**：新地图不是旅游点，而由旧案、资源、亲缘或封印节点驱动；地图越远，涉及的组织层级和超凡规则越高。

**剧情落点**：全书。证据摘要：章节索引显示丹阳、京城、三江口、火凤谷、北周、烟波城、天南、麒麟洞依次展开，原文各阶段均有对应主线事件。

**代价与限制**：离开熟悉地盘会丧失官府保护和信息优势，谢尽欢必须不断重建盟友与身份解释。

**人物和势力接口**：连接谢父旧案、凤羽草、郭太后、血雨楼、七情丹、百虫坞、尸祖终局。

**阶段变化**：地方命案扩到朝堂复仇，再到江湖大会、敌国权力、南疆灾害和天地封印。

**复用边界**：每张地图要带走前一阶段未解债务；本书几乎每次转场都由旧债或新证据推动。

**写作用途**：地图扩张和主线危险同步升级，读者不会觉得换地图是在拖篇幅。

## 4. 天地灵韵、人皇鼎与天地同寿

**设定定义**：终局阶段的天地灵韵、人皇鼎、天地同寿构成高于宗派和朝廷的世界规则层，决定尸祖能否吞并天下与强者能否以身补天。

**运行规则**：天地灵韵可被高阶存在争夺、借用或污染；人皇鼎代表人间秩序锚点；天地同寿让强者与天道相合以换取压制力。

**剧情落点**：第596-624章。证据摘要：第596章“赤血潮生如狱”、第621章“天地同寿”、第623章“不可战胜之敌”、第624章“龙吟万古”集中展示终局规则。

**代价与限制**：借天地之力会削弱自我边界，女武神以天地同寿对抗尸祖时承受被规则吞没的代价。

**人物和势力接口**：连接郭太后、墨魂生、谢尽欢、镇妖陵、麒麟洞、终局七巨头秩序。

**阶段变化**：前期通过镇妖陵和冥寂暗示世界规则，终局正式呈现天地层面的争夺。

**复用边界**：宇宙规则要有人的代价作锚；本书用郭太后的自我风险和谢尽欢的情感拉回承载抽象概念。

**写作用途**：最终战从武力胜负升到世界归属，能提高长篇终局的厚度。

## 5. 七巨头战后分责

**设定定义**：尸祖之战后，郭太后、黄麟真人、无心和尚、陆无真、谢尽欢、栖霞等高阶存在共同承担未来世界秩序，形成战后七巨头格局。

**运行规则**：胜利后不是由一个主角统治，而由不同强者按宗派、朝廷、封印、传承、外敌方向分担职责。

**剧情落点**：第627-649章。证据摘要：第627章“一个都不能少”后进入战后修复与人员分责，第647-649章以大婚和未来路收束。

**代价与限制**：每个巨头都要让渡私人自由，持续镇守世界漏洞；谢尽欢的大婚也不是退场，而是在守界前建立人间锚点。

**人物和势力接口**：连接镇妖陵修复、步青崖接续、各宗派传承、终章婚礼。

**阶段变化**：早期各强者分属不同势力，中后期在尸祖压力下协作，战后被重新组织成长期防线。

**复用边界**：分责要落到已有角色身上；临时冒出的守护者无法承接读者情感。

**写作用途**：大战后安排治理结构，能避免“打完大怪天下自动太平”的虚空感。
