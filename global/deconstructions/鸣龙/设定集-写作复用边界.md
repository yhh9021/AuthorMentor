# 设定集-写作复用边界：《鸣龙》

## 可复用结构

- **喜剧误会承载高危秘密**：给主角一个不能公开的真实原因，再设计旁观者能自洽解释的假原因，让假原因在后续成为资源或风险。
- **单元案件串主线黑手**：为每个单元案设计独立受害者和独立反派，同时让道具、术法或利益链共享一个上级来源。
- **危险外挂不代打，只提供异常接口**：给外挂一个极强领域和一个硬限制，让它在侦查、教学或情感层面高价值，在正面战斗上留缺口。
- **副作用丹药制造爽点后的债务**：先明确丹药能解决哪一个问题，再安排它制造至少一个新问题，并让新问题进入人际关系。
- **官场复仇死于制度忌讳**：先建立权力结构中的红线，再让主角诱导敌人踩线，最后由结构本身执行惩罚。
- **资源缺口驱动地图扩张**：每张新地图先给一个无法在原地解决的资源，再给资源所有者一段必须处理的人物关系。
- **情感线作为设定接口**：为每位核心情感对象配置独立专业领域和无法替代的剧情接口。
- **流言先于真相抵达**：在每个大事件后安排一个可传播的表象，再让下一阶段人物依据表象做错或做对判断。
- **强者对局先写规则缺口**：给强者配置不可硬破的领域，再给主角团分配寻找规则缺口的任务。
- **反派链条递进而非替换**：让低阶反派至少留下一种上级痕迹：供词、法器、药材、阵法、组织称号或保护伞。
- **开篇地点回收为终局责任**：在开篇设置一个无法理解的核心场所，终局让主角以全新能力和关系回到那里完成修复。
- **终局把私人情债升级为天地命题**：在中期先把情绪规则化，终局再让情感锚点承担救援、破局或自我保全功能。

## 不可复用边界

- **镇妖陵醒来：主角把自己当受害者，实际是封印事故的中心**：失忆信息必须在早期兑现一部分规则，否则会变成单纯拖谜；本书用夜红殇第一个现身把谜题转成行动约束。
- **阿飘牌显微镜：把鬼魂外挂降格成侦查接口**：外挂视野要有保密和误解成本；如果鬼魂随意公布线索，衙门体系和人际戏都会失效。
- **祸水东引：用伪造血煞把追杀者引向更大的怪物**：伪造线索必须引向真实危险，不能只为糊弄读者；本书让太叔丹和冥神教承接了这条假线。
- **中秋赌船：药物副作用把滑头推成真英雄**：药效不能解决所有难题；本书把它限定为短期爆发，并用失控、昏迷、伤势和误读回收代价。
- **言志诗误读：逃命姿态被群众加工成忠义神话**：误读必须服务后续剧情，不能只当段子；本书让名声成为谢尽欢进入更高层博弈的通行证。
- **金楼入京：从江湖案件切进谢父旧案**：官场线要给具体制度和利益，不宜只写抽象黑暗；本书用赤麟卫、京兆府、卷宗、失职处分来落地。
- **李公浦体面：不用刺杀，用皇权忌讳杀官场敌人**：政治陷阱必须提前铺设权力忌讳；没有皇帝、太子、权臣关系作支点，类似桥段会显得生硬。
- **风灵月影宗真相：三年空白不是空白，是被封存的修行地狱**：后置训练不能否定前文挣扎；本书让谢尽欢早期仍会受伤、失控、靠队友，避免回忆一开全盘变弱。
- **七星钉与焚仙蛊：用互相握命写江湖盟约**：互相握命的设定要有技术规则；本书用七星钉、焚仙蛊、火上浇油丸、栖霞记忆分别承担机制细节。
- **三江口英雄会：把江湖名声变成可争夺的公共秩序**：公共场面需要提前积累人物和议题；若没有丹阳、京城名声垫底，英雄会无法承载谢尽欢的跃迁。
- **火凤谷与朱陵火府：资源争夺背后藏着世界封印节点**：遗迹要和人物私怨相互咬合；只给古老名词会变成设定堆砌，本书用南宫、步月华、凤羽草把它落地。
- **郭太后入局：救命恩人、政治谣言和北周权力合成同一人**：流言兑现不能只靠反转身份，还要补齐当年救命事实和当前政治代价；本书两者都给了行动证据。
- **血雨楼与山海龙吟：杀手组织不只追杀，还重写亲缘和继承**：黑暗组织不宜只堆冷酷称号；本书让步青崖、沈金玉、傀儡术等具体人物和技术承担情绪重量。
- **烟波城与七情丹：把情欲喜剧推进成心性与冥寂规则**：情欲题材容易滑向重复调情；本书通过七情丹、冥寂、圣女身份把情绪纳入规则，抬高了戏剧功能。
- **天南百虫坞：低阶蛊灾最终汇入尸祖还魂**：样本案件要各自好看，不能只为伏笔存在；丹阳、火凤谷、烟波城各自有独立冲突，所以汇总时有重量。
- **麒麟洞终局：人间情债对抗万物归寂**：多线终局要提前埋接口；本书靠镇妖陵、冥寂、天地灵韵、女武神、鸣龙枪分担不同层级，避免全压主角一句爆发。
- **镇妖陵与锁魂咒**：封印规则必须让角色行动受限；若封印只当背景名词，就无法支撑主角保密和回归。
- **正伦剑与红伞神念**：法器不能万能；本书让它强在神魂视野和隐藏身份，而非直接替谢尽欢打穿所有敌人。
- **武道品级与登仙门槛**：境界台阶要和具体战术配套；只喊品级会削弱战斗可信度。
- **龙系自创武学**：命名招式要在场面中承担战术功能，不能只当口号；本书用不同招式应对围攻、横击、追击和破阵。
- **蛊毒派、尸蝇与疯尸花**：蛊毒不能只有恶心感；本书不断给出传播方式、材料来源和破解接口。
- **妖道血祭与夺元妖术**：血祭要有组织物流和遮掩成本；本书通过赌船、人群、教众、阵法来支撑。
- **冥寂与鬼修尸祖**：抽象大道要先在局部事件中展示效果；本书在七情丹和神殿段先让读者见到情绪被规则化。
- **道行暴涨丹**：不能频繁无成本使用；本书将其作为丹阳危机的极端选择，而非日常战法。
- **甲子莲、凤羽草与高阶药材**：资源线要绑定具体病人或旧债；脱离人物，药材名称会变成清单。
- **七星钉与焚仙蛊**：双向制衡必须让两边都有筹码和伤口；单向下毒无法产生同等张力。
- **八方通明阵与钦天监预警**：预警系统不能全知；本书让它受到信息差和政治干扰限制。
- **赤麟卫、京兆府与大乾律**：制度戏需要具体职位和流程；本书用捕快盘问、卷宗、失职、体面处置落地。
- **宗派镇守与监正之争**：宗派矛盾要有具体利益点；本书用监正、资源、地盘和话语权支撑。
- **地图层级：丹阳到麒麟洞**：每张地图要带走前一阶段未解债务；本书几乎每次转场都由旧债或新证据推动。
- **北周女武神与太后身份**：复合身份要分阶段揭开；若一开始全盘公开，谣言伏笔和政治张力都会消失。
- **风灵月影宗幻境训练**：训练不能抹掉现实成长；本书让谢尽欢在现实仍要靠队友、药物和策略补足境界差。
- **鸣龙枪与白圣精金**：题名武器要有功能和来历；只有名字响亮而无规则，会显得空泛。
- **天地灵韵、人皇鼎与天地同寿**：宇宙规则要有人的代价作锚；本书用郭太后的自我风险和谢尽欢的情感拉回承载抽象概念。
- **七巨头战后分责**：分责要落到已有角色身上；临时冒出的守护者无法承接读者情感。
- **喜剧误会承载高危秘密**：误会若不影响剧情，只会变成段子堆叠；本书让误会提供通行证、追查压力和政治遮掩。
- **单元案件串主线黑手**：若早期案件只靠后期解释硬连，会显得牵强；本书用蛊毒、血煞、冥神教名号反复留痕。
- **危险外挂不代打，只提供异常接口**：限制若只靠作者口头声明，读者不会接受；本书用真身封印、保密风险和镇妖陵追查让限制具体化。
- **副作用丹药制造爽点后的债务**：副作用若只在战后昏迷一句带过，会削弱规则感；本书让失控行动被旁人亲眼看见。
- **官场复仇死于制度忌讳**：没有红线铺垫时，皇帝出手会像作者代劳；本书在京城卷反复铺设储位、权臣和皇权脸面。
- **资源缺口驱动地图扩张**：只为变强而找资源会使地图工具化；本书用南宫、步月华、林紫苏、丹王共同承载情绪和利益。
- **情感线作为设定接口**：若所有女性只承担同一种情绪功能，会迅速重复；本书通过职业、势力和规则差异分层。
- **流言先于真相抵达**：流言不能总是夸主角；要同时带来威望、污名和危险，本书三者都有。
- **强者对局先写规则缺口**：缺口若太晚揭示，会像临场作弊；本书在火凤谷、烟波城、天南阶段持续铺设终局接口。
- **反派链条递进而非替换**：反派若只留下谜语，会削弱单元满足感；本书让每阶段都有明确击败对象和阶段收益。
- **开篇地点回收为终局责任**：回收若只是地名重复，力度不足；本书回收的是封印规则、夜红殇真身、谢尽欢记忆和世界责任。
- **终局把私人情债升级为天地命题**：若终局才声称爱能破局，会显得廉价；本书从七情丹和冥寂阶段就把情绪纳入规则。
